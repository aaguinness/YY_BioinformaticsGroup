BLAST: identifying genes encoded by 6 differentially expressed transcripts¶

  1. The Six transcripts are found in the in the folder: megablast_hits file: uniquetranscripts.fasta
  2. We did a standard megablast using the nucleotide database for each of the 6 transcripts.
  3. Megablast hit results were downloaded for each transcript and are found in the folder: megablast_hits
  4. We wrote a shell script to select the top hit from each hit table found in the folder megablast_hits called topHits.sh
  5. The shell script wrote a .fasta file called topHits in the folder megablast_hits

Search the NCBI protein database for amino acid sequences corresponding to these 6 transcripts.¶

  1. Protein searched for each of the top hitprotien names in the .fasta file called topHits in the folder megablast_hits. Made sure to filter proteins for animals only and selected 10 protien sequences from closely related animals.
  2. These 6 .fasta files were saved in the folder called NCBI_protien_search_results

Translate RNA Seq Data¶

  1. We wrote a python file called Translate_Updated_Final.py located in the folder called RNA_seq_translation
  2. For each .fasta file in the directory the file translates the first reading frame starting at the start codon and ends at the stop codon. The translated sequences are thwn written to new .fasta files.
  3. See notes in the python file for its specific use.

HMM modeling¶

For the HMM modeling portion we created a bash script called proj_hmm_final.sh located in the folder called NCBI_protien_search_results.

  1. This shell script first aligned the downloaded .fasta protein sequences from the NCBI the original sequences found in NCBI_protien_search_results and then saved new aligned .fasta files to the subfolder NCBI_protien_search_results/align
  2. The script then constructed 6 HMM protien models with hmmbuild and stored the .hmm models in the subfolder NCBI_protien_search_results/hmm
  3. The script then hmmsearched the 4 translated RNAseq .fasta files in RNA_seq_translation with each of the 6 HMM protein models in NCBI_protien_search_results/hmm.
  4. For each model a sequence hits file and a sequence hits table were generated and stored in the subdirectory NCBI_protien_search_results/hits_table.

The python file to make the graphs is called expressionLevels.py and is found in the directory expressionLevels. The python file builds its graphs with tabular data in the subfolder NCBI_protien_search_results/hits_table

Further exploration 1¶

We ran both a discontiguous megablast and blastn on the first and second transcript from the uniquetranscripts.fasta file. Results of the discontiguous megablast and blastn are found in the folder furtherExploration1

  • Our original megablast contained 106 hits for Transcript 1 and 55 hits for Transcript 2.
  • The discontiguous megablast resulted in 105 hits for Transcript 1 and 150 hits for Transcript 2.
  • The blastn search resulted in 106 hits for Transcript 1 and 158 hits for Transcript 2.

We realized that the broadened searches generated essentially the same hits for Transcript 1 but much more hits for Transcript 2. However, both broaded searches generated very similar results.

When we restricted the database for only Human matches. The number of hits drastically decreased for both searches. However, blastn generated more hits than discontiguous megablast. This makes sense because blastn is better suited for searching across phylogenies.

Further exploration 2¶

What would happen if we built an HMM model using more distatly related mamalian and non-mamalian organisms?

We believe that the E values would increase, and become less signifigant the more distantly related the organism was to the mouse. This would mean that the proteins from the distantly related organisms are less similar than more closely related organisms. To test our hypothesis we used two of the protiens previously examined: LHX2 and SYNPR. For SYNPR we used the NCBI proetin database to search for and gather transcripts from bovine animals and fish. For LHX2 we again searched for and gathered avian and insect LHX2 transcripts. We then built 4 additional HMM models using the same methods as above, ran an HMMsearch on all 4 of the translated RNA-seq files, and finally compared the results of the HMMsearch to the results generated by our previous mouse Hmm models. The transcripts,script, alignments, and HMM models are all found in the folder further_exploration2

For SYNPR we found:

  1. Bovine matched obese2 and had E-values of 3e-182
  2. Fish matched obese2 and had E-values of 4.1e-120
  3. The Original mouse and rat transcripts matched obese2 and had E-values of 2.5e-202

For LHX2

  1. Insects matched obese2 had E-values of 1.6e-105
  2. Birds matched obese2 had E-values of 3.1e-242
  3. The original mouse + rat transcripts mathced obese2 had E-values of 1.8e-288

For both of these proteins, even though they are fairly conserved and provide hits with obese2, the hmm models from more closley related organisms better match the translated RNAsequence data and have lower, more signigigant E-values. Our hypothese was thus correct, more distanly related organisms have higher E-values because they don't match the mouse transcript as well.
